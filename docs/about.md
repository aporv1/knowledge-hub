# About

Knowledge Hub is a working notebook by **Aporv**, an independent consultant in
**AI and cybersecurity**. It's where I turn what I'm learning into clear,
answer-first explanations — the page I wish I'd found when I was figuring the
topic out myself.

Two threads run through it:

- **AI** — concepts, techniques, models, and tools, explained in plain language
  (what the Model Context Protocol is, how context windows work, how to actually
  build things).
- **Security** — defensive and offensive fundamentals, from alert triage and
  incident response to recon and pentest methodology.

Everything here is written to be useful on first read: a short definition up top,
then the why and the how, then the misconceptions worth clearing up.
